

环境：上海星研公司的sun es59pa单片机实验箱、星研集成开发软件

资料位置：
	1、使用方法可查看“使用方法及注意事项.doc”文件
	2、程序的代码在test_routine文件夹中，编译后好的.hex/.dob也在test_routine文件夹中。

项目概述：
    设计了一个单片机实验箱故障检测系统，能对实验箱里的led、按键、TLC549、温度传感器等实验常用的21个
器件进行检测。做实验的老师或同学可以将这个软件系统烧录到实验箱的单片机里，操作者可以依照需求在21个器件
进行单一或全面检测，程序运行会通过串口在PC机上打印相应器件，选择要测试的器件后会提示接线方法。通过这个
系统，可以提高实验效率；可以让实验者减少调试耗费的时间；实验室管理者也可以通过这个软件系统进行快速找到
损坏的器件，然后进行器件更换，为后续实验提供便利。
